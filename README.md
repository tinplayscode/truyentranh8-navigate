### Installation

Truy cập [chrome://extensions/](chrome://extensions/) và chọn "Load
unpacked extension...". Sau đó chọn thư mục vừa giải nén. Sau đó bạn có thể chỉnh sửa thiết lập extension bằng cách click vào icon trên taskbar.
